AddUser2Shares
**************
Version 1.3 du 15/03/2020 (Voir l'historique à la fin du fichier).


DESCRIPTION :
=============
AddUser2Shares.cmd est un utilitaire qui permet :
          - L'ajout d'un utilisateur administrateur avec mot de passe
          - L'ajout des droits nécessaires aux différents partages disponiblent.
          - L'ajout des informations d'identifications (credentials) pour un poste client.

AddUser2Shares nécessite l'utilisation de l'utilitaire Microsoft "subinacl.exe" à placer dans le
sous dossier \Res   (https://www.microsoft.com/en-us/download/confirmation.aspx?id=23510)

	+---AddUser2Shares x.x
	          AddUser2Shares.cmd
	          LisezMoi.txt
	          +---Res
	                    subinacl.exe


UTILISATION AddUser2Shares :
===========================
Avant d'utiliser AddUser2Shares.cmd, éditez le et modifiez les paramètres de configuration ci-dessous :

      User=AdminHSF		; Nom de l'utilisateur.
      Passwd=HenrySchein37		; Mot de passe de l'utilisateur
      FullName=Administrateur HSF	; Nom complet de l'utilisateur
      Server=Serveur			; Nom du serveur (il sera demandé avant validation)

Lancez le AddUser2Shares, puis choisisez les actions à réaliser.
